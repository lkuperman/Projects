PROJECT TITLE: Maze
PURPOSE OF PROJECT: This project creates a graphical, interactive, underwater maze which allows the user to collect points, fuel, and avoid obstacles to make it through all 3 levels of the maze.
DATE: 5/7/17
AUTHORS: Leah Kuperman
LANGUAGE: Java
USER INSTRUCTIONS: Download all files and the most recent version of Greenfoot. The user must have the Java language (8.0) or newer available in order to be able to run this project. Once Greenfoot is downloaded, the user should import all of the files into a new project and run it.
